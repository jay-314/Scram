Tools Utilized:
1. Unity, C#
2. Photon Bolt for Multiplayer
3. Steamworks for microtransactions and backend services.
4. Digital Ocean and GCP for Dedicated Servers (Not Active in this Demo)

Core Features of the Project:
1. Client-Server authoritative servers
2. Client-side prediction and reconciliation for player movement
3. Lag compensation for weapons and hitboxes
4. Customizable cosmetics and clothing for your character avatar
